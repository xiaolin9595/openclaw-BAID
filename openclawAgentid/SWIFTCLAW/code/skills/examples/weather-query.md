# Weather Query Skill

## 使用场景
当用户需要查询某个城市的天气信息时，使用此 Skill 获取当前天气和未来几天的天气预报。

## 工作流程
1. 从用户输入中提取城市名称
2. 使用 web_search 工具搜索该城市的天气信息
3. 整理天气数据，包括：
   - 当前温度
   - 天气状况（晴、多云、雨等）
   - 湿度
   - 风速
   - 未来几天的预报
4. 以友好的格式返回天气信息

## 参数说明
- `city`: 必填，要查询天气的城市名称（如：北京、上海、纽约）
- `days`: 可选，预报天数，默认为 1 天，最多 7 天

## 返回格式
```
🌤️ {城市} 天气信息

📍 当前天气：
   • 温度：{温度}
   • 状况：{天气状况}
   • 湿度：{湿度}
   • 风速：{风速}

📅 未来预报：
   {日期1}: {天气} {温度范围}
   {日期2}: {天气} {温度范围}
   ...
```

## 示例

### 输入
```
查询北京的天气
```

### 输出
```
🌤️ 北京 天气信息

📍 当前天气：
   • 温度：15°C
   • 状况：多云
   • 湿度：45%
   • 风速：3级

📅 未来预报：
   今天: 多云 10°C~18°C
   明天: 晴 8°C~20°C
   后天: 晴 9°C~22°C
```

## 注意事项
- 如果城市名称不明确，可以询问用户确认
- 搜索时使用中文城市名+天气关键词，如"北京天气"
- 如果搜索结果不包含完整信息，可以搜索更具体的关键词
